 
## CityExplorerBD (React Auth Project)

**Using**

 - React
 - React Router Dom
 - Bootstrap
 - Fontawesome
 - Firebase

**Features**

 - Login with Facebook
 - Login with Google
 - Signup with Email and Password
 - Login with created credential
 - Protected Rout
 - Redirect after Authentication
